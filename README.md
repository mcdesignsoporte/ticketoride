# Ticketoride · Simulación Teatro de la Paz

Demo lista para simular compra de asientos numerados del **Teatro de la Paz, S.L.P.** con:

- interfaz completa en español
- layout por sección: **Planta Baja, Mezzanine y Balcón**
- asientos numerados por fila y butaca
- precios por sección
- checkout demo
- boletos digitales con QR visual de simulación
- validación manual en módulo de acceso
- panel con métricas de ocupación y ventas
- base compatible para publicar en **GitHub Pages** y evolucionar a **WordPress**

## Precios configurados
- Planta Baja: **$1,000 MXN**
- Mezzanine: **$750 MXN**
- Balcón: **$350 MXN**

## Cómo probar
1. Abre `index.html`
2. Entra a **Ver simulación**
3. Selecciona asientos
4. Completa el checkout
5. Revisa **Mis boletos**
6. Valida un folio en **Acceso**
7. Consulta métricas en **Panel**

## Notas
- Esta versión usa `localStorage`, ideal para demo y pruebas en GitHub.
- El archivo `js/wp-bridge.js` queda conservado como base para una integración posterior con WordPress.
- El plano cargado se usa como referencia visual del recinto.
