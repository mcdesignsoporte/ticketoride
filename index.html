Todo el frontend está pensado para que después puedas convertirlo a WordPress sin rehacer la experiencia visual desde cero.

Catálogo y detalle

Páginas pensadas para mostrar evento, sede, fecha, zonas, precios, mapa y preguntas frecuentes.

Checkout y tickets

Formulario del comprador, resumen del pedido, cupones demo y cartera digital de boletos con QR visual.

Operación de acceso

Incluye vista de scanner/check-in y panel resumen para mostrar cómo se administraría el evento.

Eventos destacados

Visual listo para presentar al cliente

Preparación para WordPress

Diseñado para que la migración sea más ordenada

El paquete incluye un puente JavaScript y snippets PHP de ejemplo para registrar eventos, exponerlos vía REST, y localizarlos al frontend. Así puedes arrancar visual ahora y conectar lógica real después.
